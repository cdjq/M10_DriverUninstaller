# DFRobot_DriverUnInstaller v1.0.0

## 文件说明
- `DFRobot_DriverUnInstaller_v1.0.0.zip`：给用户下载的安装包（内含 exe）
## 适用系统
- Windows 10/11
## 使用步骤
1. 下载并解压 `DFRobot_DriverUnInstaller_v1.0.0.zip`
2. 右键 `DFRobot_DriverUnInstaller.exe`，选择“以管理员身份运行”
3. 点击“扫描”
4. 点击“一键删除”完成驱动清理

## 注意事项
- 删除操作需要管理员权限
- 建议先断开设备，再执行清理
